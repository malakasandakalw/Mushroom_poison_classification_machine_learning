Classify Whether Mushroom is Edible or Poison application was developed on python flask framework.
Python 3.12.1 was the initial development environment.

Install required libraries

	1. flask
	2. pandas
	3. pickle
	4. numpy
	5. sklearn

Run application

	1. Open command line and navigate to folder where “app.py” is located. (default location “App/prediction-app”).

	2. Run command “Python app.py”.

	3. Application will be start and running on “http://127.0.0.1:5000”.

	4. Open any internet browser app and go to “http://127.0.0.1:5000”.

	5. Enter data that required for prediction and press submit button to check result.
